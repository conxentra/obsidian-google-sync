## Task 2: Fix browser-compatible `setTimeout` warning

**Objective:** Replace ambiguous global timer usage in runtime-compatible code with `window.setTimeout` where available.

**Files:**

- Modify: `src/google/http.ts`
- Inspect/modify if still present: `headless/authorize.ts`

**Implementation guidance:**

In `src/google/http.ts`, replace:

```ts
const sleep = opts.sleep ?? ((ms: number) => new Promise((r) => setTimeout(r, ms)));
```

with browser-safe code:

```ts
const sleep =
    opts.sleep ??
    ((ms: number) =>
        new Promise((resolve) => {
            if (typeof window !== "undefined" && typeof window.setTimeout === "function") {
                window.setTimeout(resolve, ms);
            } else {
                globalThis.setTimeout(resolve, ms);
            }
        }));
```

If `headless/authorize.ts` uses `setTimeout`, either:

- switch to `globalThis.setTimeout` if it is Node-only headless code, or
- use the same safe window/global fallback if code is shared with runtime.

**Verification:**

```bash

npm run build
```

Expected: pass.

## Task 4: Fix Buffer/string concatenation warning in headless CLI

**Objective:** Remove scorecard/type warning at `headless/cli.ts:124` by handling stdin chunks explicitly.

**Files:**

- Modify: `headless/cli.ts`

**Implementation guidance:**

Current pattern:

```ts
let data = "";
process.stdin.setEncoding("utf8");
process.stdin.on("data", (c) => (data += c));
```

Use explicit string conversion:

```ts
let data = "";
process.stdin.setEncoding("utf8");
process.stdin.on("data", (chunk: string | Buffer) => {
    data += typeof chunk === "string" ? chunk : chunk.toString("utf8");
});
```

Because `setEncoding("utf8")` makes chunks strings at runtime, this is mostly scanner/type hygiene.

**Verification:**

```bash

npm run test:unit
```

Expected: pass.

## Task 6: Prove headless files are not bundled into `main.js`

**Objective:** Add a lightweight regression check to prevent accidental Node-only imports from entering runtime bundle.

**Files:**

- Modify: `package.json`
- Create: `scripts/check-runtime-bundle.mjs`

**Implementation guidance:**

Create `scripts/check-runtime-bundle.mjs`:

```js
import { readFileSync } from "node:fs";

const bundle = readFileSync("main.js", "utf8");
const forbidden = [
    "node:fs",
    "node:path",
    "node:process",
    "node:child_process",
    "node:http",
    "headless/",
];

const hits = forbidden.filter((needle) => bundle.includes(needle));
if (hits.length) {
    console.error(`Runtime bundle contains forbidden Node/headless references: ${hits.join(", ")}`);
    process.exit(1);
}
console.log("Runtime bundle has no Node/headless references.");
```

Add package script:

```json
"check:runtime-bundle": "npm run build && node scripts/check-runtime-bundle.mjs"
```

**Verification:**

```bash
npm run check:runtime-bundle
```

Expected: pass.
